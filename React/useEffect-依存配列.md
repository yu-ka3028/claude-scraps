**Q:** `useEffect` の依存配列は何を制御する？`[]`・`[a, b]`・省略の違いは？

**A:** エフェクトの再実行タイミングを制御する。
- `[]`: マウント時のみ実行
- `[a, b]`: `a` か `b` が変化したときに再実行
- 省略: レンダーのたびに毎回実行

```js
useEffect(() => { fetch('/api/user').then(setUser); }, []);                               // [] マウント時のみ
useEffect(() => { const c = connect(url, id); return () => c.disconnect(); }, [url, id]); // 変化時に再実行
useEffect(() => { console.log(count); });                                                  // 毎回
```

**メモ:** 依存配列に含めるべき値は「エフェクト内で参照している props・state・コンポーネント内変数」。値が変わるたびに前のエフェクトのクリーンアップが走ってから新しいエフェクトが実行される（[[useEffect-クリーンアップ関数]] 参照）。
