**Q:** Reactのマウント・アンマウントとは？

**A:** マウント = コンポーネントが画面に初めて表示されDOMに追加される瞬間。アンマウント = 画面から消えDOMから取り除かれる瞬間。

**メモ:** ライフサイクルの流れ：マウント（エフェクト実行）→ 再レンダー（依存配列変化時にエフェクト再実行）→ アンマウント（クリーンアップ実行）。[[useEffect-依存配列]] [[useEffect-クリーンアップ関数]] と組み合わせて理解する。
