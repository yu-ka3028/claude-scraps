**Q:** Reactのフックがループや条件分岐の中で使えない理由は？

**A:** ReactはフックをN番目の呼び出しという順序で管理しているため、毎レンダリングで同じ順序・同じ回数呼ばれる必要がある。ループや条件分岐では呼び出し回数が変わる可能性があるためNG。

**メモ:** 解決策はループの中身を別コンポーネントに切り出すこと。`List` が何個 `Item` をレンダリングするかは変わってOK。各 `Item` の内部は常に `useState` を1回だけ呼ぶため順序が崩れない。

```jsx
function Item({ item }) {
  const [selected, setSelected] = useState(false); // 常に1回
  return <div>{item}</div>;
}
function List({ items }) {
  return items.map(item => <Item key={item.id} item={item} />);
}
```
