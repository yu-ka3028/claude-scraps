**Q:** Reactにおけるpropsとstateの違いは？

**A:** propsは親から渡される値で自分では変えられない。stateは自分が`useState`で管理する値で自分で変えられる。

**メモ:** useEffectでは「propsが変わったら外部システムも更新する」パターンが多い。[[React/useState-set関数-更新関数.md]] はstateの更新を扱っている。
