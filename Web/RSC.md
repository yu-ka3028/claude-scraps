**Q:** React Server Components（RSC）とは何か？従来の SSR と何が違うか？

**A:**
RSC はサーバー上でのみ動作する React コンポーネント。
- JS をブラウザに送らない（バンドルサイズが減る）
- hydration（イベントリスナーのアタッチ）が不要
- DB や API に直接アクセスできる

従来の SSR との違い:

| | SSR（従来） | RSC |
|---|---|---|
| JS 送信 | あり | なし |
| hydration | あり | なし |
| DB アクセス | API 経由 | 直接可 |
