`Scraps/` ディレクトリ内の全カテゴリファイルを読み込み、今日の復習状況を表示する。

## 手順

1. `Scraps/` 内のMarkdownファイルを全て読み込む（以下は除外）:
   - `CLAUDE.md`
   - `_template.md`
   - `_archive.md`

2. 各スクラップの「次回復習日」を確認し、以下を集計する:
   - **本日の復習対象**: 次回復習日 ≤ 本日 のスクラップ数
   - **うち持ち越し**: 次回復習日 < 本日（過去日付）のスクラップ数

3. 以下の形式で表示する:

```
YYYY-MM-DD のスクラップ状況
本日の復習対象: N件（うち持ち越し M件）
```

復習対象が0件なら「今日の復習対象はありません」と表示する。
