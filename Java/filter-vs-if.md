**Q:** JavaのStream `.filter()` と `if` 文はどう使い分ける？

**A:** Streamチェーン内で要素を絞り込むなら `.filter()`、単一値への分岐・早期リターンは `if`。Streamの中で要素を「通過させるか除外するか」を決めるのが `.filter()` の役割で、`if` の代替ではなくStreamに特化した絞り込み手段。

**メモ:**
- `.filter()` のラムダ内で外部変数を参照する場合、effectively final（実質変更されない）でないとコンパイルエラー
- `.forEach()` や `.map()` のブロック内に `if` は書けるが、それは絞り込みではなくスキップ処理。意図が不明確になるので `.filter()` を使う方が良い
