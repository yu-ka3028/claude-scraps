**Q:** Javaで`int`と`Integer`のnull・デフォルト値の違いは？

**A:** `int`はプリミティブ型でデフォルト値は`0`、nullを許容しない。`Integer`はラッパークラス（オブジェクト）でnullを許容し、フィールドのデフォルト値は`null`。

**メモ:** この「プリミティブ vs ラッパー」の二重構造はJava・C#特有。KotlinやSwiftは`Int`/`Int?`で統一的に表現。RubyやTypeScriptはすべてがオブジェクトなのでプリミティブ型自体が存在せず、この区別がない。[[Java/NotBlank-アノテーション.md]] と同じくJavaの型システムに関わるトピック。
