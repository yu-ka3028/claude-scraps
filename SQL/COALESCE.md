**Q:** COALESCE は何を返す関数か？

**A:** 引数リストを左から評価し、最初に NULL でない値を返す。全て NULL なら NULL を返す。

**メモ:** EAVアンチパターンで登場しやすい。EAVでは動的な属性を行として持つため、型別カラム（`attr_string`, `attr_int` など）から値を取り出す際に `COALESCE(attr_string, attr_int)` のような後始末が必要になる。
